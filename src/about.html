What Is De-Slop?

De-Slop is a Chrome extension that detects and removes AI-generated filler content from web pages. It uses a three-tier weighted pattern matching system to score text and flag content that reads like it was written by a language model on autopilot.

It runs entirely in your browser. No servers, no API calls, no data collection. You install it, configure your sensitivity, and it works.
