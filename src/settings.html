Pattern Help

Regex Format

Patterns use JavaScript regular expression format: /pattern/flags

/\bdelve into\b/gi Matches "delve into" as whole words, case-insensitive
/\b(synergy|leverage)\b/gi Matches either "synergy" or "leverage"
/^I'm excited to/gim Matches lines starting with "I'm excited to"

Plain Text Mode

In plain text mode, type a phrase like delve into and it auto-wraps as /\bdelve into\b/gi

Word boundaries (\b) ensure partial words are not matched.

Common Regex Elements

\bWord boundary
.Any character
+One or more
*Zero or more
?Optional
(a|b)Either a or b
[abc]Any of a, b, c
^Start of line
$End of line
gGlobal (find all)
iCase insensitive
mMultiline
