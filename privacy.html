De-Slop Privacy Policy

Last updated: January 2025

TL;DR: De-Slop does NOT collect, store, or transmit any user data.

Overview

De-Slop is a browser extension that detects and removes AI-generated content from web pages. The extension operates entirely on your local device.

Data Collection

De-Slop does NOT collect, store, or transmit any user data.

All content analysis happens locally on your device. Your browsing activity and filtered content never leave your computer.

Local Storage

The extension stores the following preferences locally on your device using Chrome's sync storage:

  • Extension enabled/disabled state
  • Detection sensitivity level (1-5)
  • Detection type toggles (AI slop, corporate buzzwords, marketing spam, etc.)
  • Custom pattern configurations
  • LinkedIn fixer preferences (video blocking, darker mode, ad blocking)

These settings may sync across your Chrome browsers if you have Chrome sync enabled. This is a built-in Chrome feature and does not involve our servers.

Permissions

De-Slop requires the following permissions:

  • storage - Save your preferences locally
  • activeTab - Read page content to detect AI-generated patterns
  • sidePanel - Display the control panel as a sidebar
  • declarativeNetRequest - Block LinkedIn ad tracking (only if you enable this feature)
  • <all_urls> - Run on any website you visit to filter content

Third-Party Services

De-Slop does NOT use any third-party services:

  • ✗ No analytics or tracking
  • ✗ No external APIs
  • ✗ No remote code loading
  • ✗ No data transmission
  • ✗ No crash reporting
  • ✗ No telemetry
